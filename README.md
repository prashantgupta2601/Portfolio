# Portfolio

A modern, high-performance, and fully responsive personal portfolio website built to showcase my frontend development skills, projects, and UI/UX capabilities.

This portfolio focuses on clean design, smooth animations, and best frontend practices using the latest web technologies.

## Highlights

- Modern UI with clean and minimal design  
- Smooth animations and transitions  
- Fully responsive on Mobile, Tablet, and Desktop  
- Optimized performance and fast loading  
- Well-structured and scalable React components  
- Interactive project showcase section  
- Professional layout suitable for recruiters and clients  

## Tech Stack

- React.js  
- Tailwind CSS  
- Framer Motion (Animations)  
- JavaScript (ES6+)  
- HTML5 & CSS3  

## Project Structure

src/
│
├── components/ # Reusable UI components
├── pages/ # Page sections (Home, About, Projects, Contact)
├── assets/ # Images and icons
├── App.jsx
└── main.jsx
